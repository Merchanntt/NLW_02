## Getting Started
To use this app, you'll need...

- Clone this repository: 
```shell
cd "dir from your preference"
git clone https://github.com/Merchanntt/Proffy
``` 
- Run: 
```shell
Yarn
```
- Create the dataBase:
```shell
cd.. server
yarn knex:migrate
```
- And finally, run the servers:
```shell
cd mobile
yarn start
-
cd server
yarn dev:server
-
cd web
yarn start
```

Feel free to change on your one way <3
